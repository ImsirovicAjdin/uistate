UIstate Examples

These examples showcase different approaches to state management using UIstate's declarative patterns. Each example demonstrates a specific pattern or technique.

Range Sliders

  • Basic Range Slider - A simple slider implementation with UIstate
  • CSS-Derived Range Slider - Using CSS to derive states
  • Attribute-Derived Range Slider - Deriving states from HTML attributes
  • Domain State Slider - Slider with semantic state domains
  • Calc-Derived Range Slider - Using CSS calc() for dynamic state

Toggle Controls

  • Button Toggle - Simple toggle button implementation
  • Two-Button Toggle - Toggle with separate on/off buttons
  • Derived Toggle State - Toggle with derived visual states

Font Controls

  • Font Adjuster - Basic font size adjustment
  • Domain Font Adjuster - Font adjustment with semantic domains
  • Explicit Font Adjuster - Font adjustment with explicit state selectors
