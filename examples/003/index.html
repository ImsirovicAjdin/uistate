Demo 003: Input value projection on click

The code demonstrates:
  • An input field is marked with data-state-for="input"
      • A button, when clicked, will:
    • Execute the action handler specified by data-state-action="thisStateActionIsHandlerInRSH"
    • Take the value from the source element (the input) specified by data-state-source="[data-state-for='input']"
    • Update the state variable actualValueOfInput specified by data-state-target
    • The span element with data-observe="actualValueOfInput" specified by data-state-target.
